## EET_Micromouse
A robot controller for Micromouse. Using Non-Encoded Motors and a Arduino Uno. 

## Used Components
1. Arduino Uno
2. L298N Motor Controller
3. 2x Non-encoded Hobby Motors
4. 3x Ultra Sonic Sensors
5. IR Sensors
6. 3x 18650 3.7v Batteries
7. BMS controller

## Highlights
This project was a fantastic challenge that reinforced our practical skills in Embedded Systems, Solidwork modeling, and Troubleshooting. It was a testament to the power of effective collaboration, especially during the three week long debugging sessions.
